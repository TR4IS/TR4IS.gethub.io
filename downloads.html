Antivirus & Windows Defender Notice

Some programs on this page apply system-level tweaks, write to Windows Registry entries, monitor file system activity, or use techniques that security software may not recognise. Windows Defender, SmartScreen, and third-party antivirus tools may flag these as threats or Potentially Unwanted Programs (PUP) — these are false positives. The programs do exactly what is described, nothing more.

  • All source code is publicly auditable on GitHub before you download
  • If flagged, add the file to your antivirus exclusions list and run again
  • Windows SmartScreen: click “More info” → “Run anyway”
  • When in doubt — scan with VirusTotal before running
