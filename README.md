# reachflow.site

Free personal pages for developers — sign in with GitHub, get your space instantly.

**Live:** [reachflow.site](https://reachflow.site)

---

## What it is

reachflow gives every developer a free, fully customizable personal webpage at `reachflow.site/spaces/{username}`. No config files, no hosting setup — sign in with GitHub and your page is live in under a minute.

## Features

- One-click GitHub OAuth signup
- Drag-and-drop layout — move your photo, name, and bio anywhere on screen
- Full customization: 25+ fonts, text shadows, animations, avatar rings, background images/videos, gradients
- Social links with platform logos (GitHub, X, Instagram, LinkedIn, YouTube, TikTok, Discord, Twitch, Reddit)
- Custom stickers, text blocks, and music player
- Owner-only editor — only you can edit your own space

## Stack

| Layer | Tech |
|---|---|
| Hosting | GitHub Pages |
| Auth + Save API | Cloudflare Workers |
| OAuth | GitHub OAuth App |
| Storage | GitHub Contents API (HTML files in `/spaces/`) |
| Frontend | Vanilla HTML/CSS/JS — zero dependencies |

## Project structure

```
/                        Landing page
/auth/callback.html      OAuth callback
/editor.html             Space editor
/spaces/{username}/      User spaces (generated by worker)
/worker/index.js         Cloudflare Worker (auth, save)
/terms.html              Terms of use
```

## Worker setup

The worker handles OAuth token exchange, session tokens (HMAC-SHA256), and saving spaces via the GitHub Contents API. Secrets are managed via Wrangler and never committed.

```sh
cd worker
wrangler secret put GITHUB_CLIENT_SECRET
wrangler secret put REPO_TOKEN
wrangler secret put TOKEN_SECRET
wrangler deploy
```
